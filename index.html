Initiation Guide ✅📑

Ideation Phase 💡

Brainstorm ideas for the application, such as the possible architecture. Mock up the UI (in MS Paint), decide on the Font, button styling, theme. Make paper mockups of the UI as well based on the specifications.

Also, list down the specified tools as per the specifications.

Initial Run-through ➡️

Run through the video/article to get an idea about the topic to implement or work on. You should re-run through the topics (An overview) in a top 🡪 bottom approach.

What is Top 🡪 Bottom Approach? Initially, you have to glance through the article/webpage/video to get an idea of what it is. You don't need to understand each nook and corner. Next, if you are implementing something, circle out the thing (which is important) and ask any AI (ChatGPT/Gemini/Copilot) to explain the topic with examples/code (if applicable), also ask for analogies for easier understanding.

Implementation Phase ⚒️

Implement the application in the platform (like VS Code, IntelliJ, or any other IDE/Text editor). Install the required libraries/dependencies. Use AI/Google on troubleshooting and to properly set up the environment.

Error Correction Phase ❌

Correct errors in code and test every single file/component and make sure everything works as intended. After satisfaction, proceed to deploy the application.

🎉 Congratulations! Now, you have deployed your application successfully.
